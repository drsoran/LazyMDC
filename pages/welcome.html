Welcome to Data Card Builder v2.

This update is mostly about cleaning up the old one and making it a little more streamlined for making future additions and flows much easier to implement

As a side effect, we've also moved to a more guided route through creating an MDC that should hopefully be far more intuitive and be able to add tooltips without it seeming to be overwhelming

You can save at any time using Ctrl+S which will update your URL should you wish to collaborate by sending it onto someone else. When you donwload a kneeboard then it'll snapshot the original so you don't need to worry about someone modifying it
